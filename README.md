## Lekki
**Project Brief:** 
The site online pharmacy where you can find medicine on the cheapest price.
**Tags**: pug, Sass, JS.

`npm i`
`npm i -g tars-cli`  
`npm run start`

**How to build the project:**
`npm run build`
